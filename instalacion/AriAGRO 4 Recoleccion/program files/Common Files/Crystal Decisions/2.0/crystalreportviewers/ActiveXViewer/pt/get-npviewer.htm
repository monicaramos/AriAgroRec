Obter Crystal Report Viewer - Plug-in do Navegador

O plug-in do Crystal Report Viewer não pôde ser localizado na máquina. Um tempo de download de aproximadamente 4.5MB é necessário para instalar o visualizador. Para continuar, clique no vínculo abaixo e, quando solicitado, salve o arquivo. No menu Iniciar, escolha Executar e digite o caminho completo do local onde você salvou o programa transferido, ou, no Windows Explorer, selecione o arquivo transferido e clique duas vezes nele. Esse procedimento instala o Crystal Report Viewer. Depois de instalado o plug-in, você deve reiniciar o Navegador para que ele possa carregar o plug-in.
