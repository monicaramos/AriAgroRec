Obtener visor de Crystal Report - Complemento para Navigator

No se encontró el complemento para el visor de Crystal Report en el equipo. Para instalar el visor deberá descargar un programa de aproximadamente 4.5 MB. Para continuar, haga clic en el vínculo que aparece más abajo y, cuando el sistema se lo pida, guarde el archivo. En el menú Inicio, seleccione Ejecutar y especifique la ruta de acceso completa a la carpeta donde guardó el programa descargado o, alternativamente, seleccione el archivo descargado en el Explorador de Windows y haga doble clic en el mismo. Así instalará el visor de Crystal Report. Una vez que haya instalado el complemento, deberá reiniciar Navigator para que éste pueda cargar el complemento.
