Navigator-Plugin zum Crystal Report Viewer nicht gefunden

Das Crystal Report Viewer-Plugin konnte auf Ihrem Computer nicht gefunden werden. Der Viewer kann mit einem einmaligen Download von etwa 4.5MB installiert werden. Um fortzufahren, klicken Sie auf den Link unten, und speichern Sie die Datei, wenn Sie dazu aufgefordert werden. Wählen Sie aus dem Menü "Start" den Befehl "Ausführen", geben Sie den vollständigen Pfad zum Speicherort des geladenen Programms ein, oder wählen Sie im Windows Explorer die geladene Datei, und doppelklicken Sie darauf. Dadurch wird der Crystal Report Viewer installiert. Nachdem Sie das Plugin installiert haben, müssen Sie Navigator neu starten, damit das Plugin geladen wird
