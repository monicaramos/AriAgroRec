Installez Crystal Report Viewer - Navigateur prêt à l'emploi

Le système n'a pas reconnu le programme Crystal Report Viewer sur votre ordinateur. Il est nécessaire de télécharger le programme (4.5 MB) pour installer le visualiseur. Pour continuer, cliquez sur le lien ci-dessous et enregistrez le fichier lorsqu'une boîte de dialogue vous y invite. Cliquez sur Démarrer, choisissez Exécuter et saisissez le nom complet du chemin où vous avez enregistré le programme téléchargé, ou cliquez deux fois sur le fichier téléchargé dans l'Explorateur de Windows pour installer Crystal Report Viewer. Redémarrez ensuite le navigateur pour qu'il puisse à son tour charger le programme.
