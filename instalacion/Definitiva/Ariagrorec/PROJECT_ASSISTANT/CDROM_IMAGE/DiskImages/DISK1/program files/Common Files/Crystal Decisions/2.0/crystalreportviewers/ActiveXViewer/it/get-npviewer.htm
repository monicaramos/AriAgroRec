Procurarsi Crystal Report Viewer - Plug-in di Navigator

Impossibile individuare il plug-in di Crystal Report Viewer nel sistema. È necessario scaricare in una sola operazione approssimativamente 4.5MB per installare il viewer. Per continuare, fare clic sul collegamento sottostante e, quando viene richiesto, salvare il file. Dal menu Avvio, scegliere Esegui e digitare il percorso completo dove è stato salvato il programma scaricato, o da Windows Explorer selezionare il file scaricato, e fare doppio clic su di esso. In questo modo si installerà il Crystal Report Viewer. Una volta installato il plug-in è necessario riavviare Navigator affinché possa caricare il plug-in.
